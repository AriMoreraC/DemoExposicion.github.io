Elementos De Importancia Para Tener Una Pagina Responsiva

Según nos da a conocer (Sotelo, 2014) hay tres ingredientes técnicos fundamentales para tener un diseño web responsivo.

Fluid Grid

Cuadricula fluida, En lugar de pixeles recomiendan porcentajes para definir los divs.

Imagenes Flexibles

Reducir con porcentajes el mínimo de la imagen para ajustarse a diferentes pantallas y resoluciones.

Media queries

Permiten consultas CSS, según el ancho mínimo y el máximo de un ordenador.
